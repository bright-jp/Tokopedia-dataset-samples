# # Tokopedia データセットサンプル

<h2>1001件のTokopedia商品のサンプルデータセット</h2>

![Tokopedia dataset header](https://github.com/bright-jp/Tokopedia-dataset-samples/blob/main/Tokopedia-datasets.png)

1000件以上のレコードを含むTokopediaデータセットのサンプルです。データセットは<b>Bright Data API</b>を使用して抽出されています。

<h2>Tokopediaデータセットに含まれるデータポイントの一部：</h2>

* ```product_id```: 商品ID
* ```title```: 商品タイトル
* ```url```: 商品リスティングのURL
* ```currency```: 価格の通貨
* ```delivery```: 商品の配送詳細
* ```final_price```: 商品の最終価格
* ```initial_price```: 商品の初期価格
* ```seller_name```: 商品の販売者名
* ```description```: 商品説明
* ```availability```: 商品の在庫状況
* ```reviews_count```: 商品のレビュー数
* ```rating_count```: 商品を評価した人数
* ```rating```: 商品評価
* ```discussion_count```: 商品に関するディスカッション数
* ```categories```: 商品カテゴリ
* ```images_count```: 商品画像数

ほかにも多数あります。

これは「Tokopedia products」データセットから派生したサンプルサブセットであり、元のデータセットには<b>120万件以上のレコード</b>が含まれます。

利用可能なデータセットのファイル形式：<b>JSON, NDJSON, JSON Lines, CSV, または Parquet。必要に応じてファイルを .gz に圧縮できます</b>。

データセットの配信タイプの選択肢：<b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>。

更新頻度：<b>Once, Daily, Weekly, Monthly, Quarterly, または Custom basis</b>。

抽出されたデータポイントに加えて利用可能なデータエンリッチメント：<b>リクエストに基づきます。</b>

<b>[Tokopediaの完全なデータセットを入手](https://brightdata.jp/products/datasets/tokopedia)</b>。

<h2>Tokopediaデータセットのユースケースとは？</h2>

<h3>1. 商品インサイト＆価格戦略</h3>
Tokopediaの売れ筋商品を把握し、在庫、価格、サプライチェーン戦略を強化できます。人気の検索や需要の高い商品に関するインサイトを活用してマーケティング施策を最適化し、市場での競争優位性を維持します。

<h3>2.  競合分析</h3>
上位の販売者の商品ラインナップ、カスタマーレビュー、プロモーション戦略を分析することで、Tokopediaのトップセラーを上回れます。新たなビジネス機会を発見し、マーケットプレイスにおけるプレゼンスを強化するためのインサイトを獲得します。

<h3>3. 消費者センチメント＆ブランドトレンド</h3>
Tokopediaでトレンドとなっているカテゴリやブランドを追跡し、消費者需要の変化を把握できます。地域ごとの商品人気を分析し、変化する購買者嗜好に合わせてブランド戦略を整合させます。
